# PDF OCR with PaddleOCR

This project extracts text from scanned PDF files using PaddleOCR.

## Features
- PDF to image conversion
- Text detection and recognition
- French OCR support
- Windows compatible

## Requirements
- Python 3.10
- Poppler (Windows)

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/pdf-ocr-paddle.git
cd pdf-ocr-paddle
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
